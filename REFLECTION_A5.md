# Reflection – Assignment 5

## Challenges Faced

One of the main challenges was converting system requirements into clear use cases. It required understanding how users interact with the system step by step.

Another challenge was ensuring that all use cases aligned with stakeholder needs from Assignment 4.

## Trade-offs

Simplifying use cases made the system easier to understand but reduced complexity in modelling advanced features.

## Learning Outcome

This assignment helped in understanding how to model user interactions and validate system behavior using test cases. It also showed how important it is to connect requirements to real user actions.

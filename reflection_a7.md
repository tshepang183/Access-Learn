# Reflection – Assignment 7

## Challenges Faced

One challenge was selecting the most suitable GitHub template. While many templates exist, not all are appropriate for Agile development. Choosing between simplicity and automation required careful consideration.

Another challenge was customizing the Kanban board. Understanding how to link issues and manage workflows took time.

## Comparison with Other Tools

Compared to tools like Trello and Jira:

- GitHub Projects is integrated with code and issues
- Trello is easier for beginners but less technical
- Jira is more powerful but more complex

GitHub Projects is ideal for developers because everything is in one place.

## Learning Outcome

This assignment helped in understanding how Agile teams manage work visually. It also improved skills in using GitHub for project management, which is important in real-world software development.
